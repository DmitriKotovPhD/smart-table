# sp6-2_smart-table_starter

## Описание проекта
Функционал веб-приложения данного проекта предоставляет поиск, сортировку, фильтрацию и пагинацию табличных данных в режиме онлайн

## Структура проекта
1. `data` - каталог с примерами данных
2. `src/main.js` - основной файл программы
3. `index.html` - файл для запуска программы
4. `components/*` - компоненты веб-приложения

## Инструкция по запуску
Открыть файл `index.html` в браузере, использовать поля управления для сортировки, поиска, фильтрации, пагинации.


## Ключевые требования
Суть организации потока данных в данном проекте: используется всего один обработчик событий, что обеспечивает скорость.
Компоненты программы добавляются согласно шаблону декоратора, аддитивно.
